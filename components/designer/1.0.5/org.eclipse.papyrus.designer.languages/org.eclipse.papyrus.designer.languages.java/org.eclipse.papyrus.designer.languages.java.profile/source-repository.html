Overview

This project uses Git to manage its source code. Instructions on Git use can be found at http://git-scm.com/documentation.

Web Browser Access

The following is a link to a browsable version of the source repository:

https://git.eclipse.org/c/papyrus/org.eclipse.papyrus-designer.git/org.eclipse.papyrus.designer.languages/org.eclipse.papyrus.designer.languages.java/org.eclipse.papyrus.designer.languages.java.profile

Anonymous Access

The source can be checked out anonymously from Git with this command (See http://git-scm.com/docs/git-clone):

$ git clone git://git.eclipse.org/gitroot/papyrus/org.eclipse.papyrus-designer.git

Developer Access

Only project developers can access the Git tree via this method (See http://git-scm.com/docs/git-clone).

$ git clone ssh://git.eclipse.org/gitroot/papyrus/org.eclipse.papyrus.tools.git

Eclipse Papyrus Software Designer project. Design your object-oriented or component-based application model and generate C++ or Java code. Reverse engineering from existing source code is also supported. State-chart code generation is provided through model-to-model transformations that are executed on an intermediate model prior to code generation. Component-based modeling is supported via the new OMG standard UCM. Deployment and configuration is possible via a deployment plan, a set of instance specifications with allocation information.
