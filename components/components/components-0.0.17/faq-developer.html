General

Where to retrieve the code?

You have to clone the git repository

git clone
					https://git.eclipse.org/r/papyrus/org.eclipse.papyrus.tools
Where is the website of the project?

To pickup the last release documentation, you have to go https://www.eclipse.org/papyrus/components/components/

Nightly - Snapshot version is available here: https://hudson.eclipse.org/papyrus/view/Components/job/papyrus-components-website/ws/site-staging/index.html

Where are the continuous integration jobs?

You have to go https://hudson.eclipse.org/papyrus/view/Components/

Where is the review server?

You have to go to gerrit: https://git.eclipse.org/r/#/q/status:open+project:papyrus/org.eclipse.papyrus.tools

Installation

How do I install Papyrus Components?

Papyrus Components is not installable: it is just a shared environment for all Papyrus components: parent pom, release script...

Check the license

How do I add automatically the license header to my java files?

mvn license:format -Dlicense.header=./components/org.eclipse.papyrus.components.parent/src/etc/epl.header.cea.txt
