About SysML Table

This plug-in contains the different modules relatives to the tables used in SysML 1.4

Project Modules

This project has declared the following modules:

Name Description
org.eclipse.papyrus.sysml14.nattable.common The plugin gather the shared API for the table based upon nattable framework.
org.eclipse.papyrus.sysml14.nattable.allocation This plugin defines the allocation table of SysML 1.4.
org.eclipse.papyrus.sysml14.nattable.allocation.tests This plugins enables unit tests on the allocation tables.
org.eclipse.papyrus.sysml14.nattable.requirement This plugins defines the requirement table of SysML 1.4.
org.eclipse.papyrus.sysml14.nattable.requirement.tests This pluygin is in charge of unit tests for requirement table.
org.eclipse.papyrus.sysml14.nattable.ui This plugin defines the user interface specific to tables in SysML 1.4.
